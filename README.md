# API TypeGraphQL

## Projeto
- API constrúida pelo TypeGraphQL

## Tecnologias
- Node.js
- TypeScript
- GraphQL
- TypeGraphQL

## Começando
```
  - Clonar o Repositório
  git clone https://github.com/estevam5s/dowhile-api-typegraphql.git

  - Instalar as Dependências
  yarn install ou npm start

  - Rode o Projeto
  yarn dev ou npm run dev
```
